# Kernel-Stop-Machine

## Overview
キャラクデバイスからシステムを停止

## Requirement
- Ubuntu20.04.2
- kernel5.8.0-53
- GCC

## How to install

HTTP
```shell
git clone https://github.com/fumiya5863/Kernel-Stop-Machine.git
```

SSH
```shell
git clone git@github.com:fumiya5863/Kernel-Stop-Machine.git
```
